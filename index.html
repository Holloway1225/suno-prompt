How it works
From Lyrics — Paste finished lyrics. The engine reads them and builds four things: title suggestions pulled from the lyric's actual language, a full sound production brief, a short Suno-ready style tag (under 1000 chars), and the same lyrics normalized with Suno section tags. No band or artist names ever.

Band Fusion — Enter two or three bands and set a blend weight on each (sliders auto-balance to 100%). Add up to 2 reference songs each. The engine deep-reads each sonic identity — genre, instrumentation, production technique, vocal style, dynamics, texture — then blends them by weight into a cohesive Suno-ready style prompt under 1000 characters. The percentages tell it which sound is load-bearing and which is seasoning. All output is pure sonic description; no real names appear anywhere.

No folk — Both flows hard-exclude folk, acoustic singer-songwriter, Americana, and country, and require a heavy/electronic/electric backbone. A post-check scans the output and flags any folk language that slips through.

Engine — Llama is fast and free. Claude reads the nuance deepest. Everything is copyable, and every run is saved to History on this device.
